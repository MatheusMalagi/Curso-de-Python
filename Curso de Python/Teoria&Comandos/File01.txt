print > significa escreva

input > significa leia

CTRL + ; = comenta o código selecionado

int > numeros inteiros
float > números reais, que tem vírgula (pode ser inteiro se for por ex: 7,0)
bool > somente aceita "True" ou "False" sempre usar com letras minusculas inicialmente
str > sempre estarão entre '' para indicar que são caracteres


ordem de procedência >
1º () - o que está no parenteses
2º ** - exponenciação
3º * / // e % - multiplicação, divisão, divisão inteira e resto da divisão, é só fazer quem aparece primeiro
4º + e - - adição e subtração.
